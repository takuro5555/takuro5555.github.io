## アプリについて

直感スピードアリーナは、

**直感とスピードで答えるシンプルなクイズゲーム**です。

2択〜4択のクイズに挑戦し、

30秒の制限時間内でどこまで正解できるかを競います。

コンボでスコアを伸ばすモードや、

ミスすると終了する緊張感のあるモードなど

複数のゲームモードを楽しめます。

また、端末内ランキングにより

家族や友達とスコアを競いながら遊ぶこともできます。

---

## 主な機能

・2択〜4択のスピードクイズ

・コンボモード / ノーミスモード

・複数のクイズパック

・端末内ランキング機能による家族や友達とスコア対決
